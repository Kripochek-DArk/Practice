# Лабораторная №4

## Задача 1

### Текст задачи
Дерево содержит целые числа. Вычесть из каждого разряда числа 1, если
разряд не содержит 0, иначе заменить этот разряд на 1. Например, 1024 → 113

### Алгоритм решения

- Считать числа с клавиатуры до ввода пустой строки.
- Проверить корректность введённых данных (число и не отрицательное).
- Сохранить введённые числа в массив.
- Построить бинарное дерево из массива:
    - левый потомок: 2*i + 1
    - правый потомок: 2*i + 2
- Реализовать функцию mapTree, которая применяет функцию преобразования ко всем узлам дерева.
- Реализовать функцию преобразования числа:
    - пройти по всем цифрам числа
    - если цифра равна 0, заменить её на 1
    - иначе уменьшить цифру на 1
- Построить новое дерево.
- Вывести исходное и преобразованное дерево.

## Задача 2

### Текст задачи
Дерево содержит строки. Сколько узлов дерева заканчивается на заданный
символ?

### Алгоритм решения

- Считать строки с клавиатуры до ввода пустой строки.
- Сохранить введённые строки в массив.
- Построить бинарное дерево из массива:
    - левый потомок: 2*i + 1
    - правый потомок: 2*i + 2
- Реализовать функцию foldTree, которая выполняет обход дерева и накапливает результат.
- При обходе дерева проверить последний символ каждой строки.
- Если строка заканчивается на заданный символ — увеличить счётчик.
- После обхода дерева вывести количество подходящих узлов.
