# Лабораторная №1

## Задача 1

### Текст задачи

Сформировать список из чисел, на 1 меньших, чем вводимые значения.

### Алгоритм решения

- в функцию передается пустой список
- считывание числа из терминала
- проверка есть ли там число
- если ничего нету, то заканчиваем рекурсию
- иначе рекурсивный вызов с списком, куда добавлено число меньше на 1 введенного пользователем и снова со 2 пункта
- вывод значений списка

### Текст задачи

Сколько строк введено? (Признак конца ввода — пустая строка)

## Задача 3
### Текст задачи

Реализовать собственные функции для выполнения основных операций над списками:  
добавление элемента в конец списка, удаление элемента, поиск элемента, сцепка двух списков и получение элемента по индексу.  

### Алгоритм решения

- создается исходный список чисел
- реализуются рекурсивные функции:
    - добавление элемента в конец список
    - удаление первого вхождения элемента
    - проверка наличия элемента в списке
    - сцепка двух списков
    - получение элемента по индексу
- пользователю предлагается:
    - ввести число для добавления в конец список
    - ввести число для удаления из списка
- выполняется проверка корректности ввода
- при корректном вводе выполняется соответствующая операция
- выводится обновленный список после каждой операции
