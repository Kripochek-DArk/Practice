# Лабораторная №3

## Задача 1

### Текст задачи

На основе последовательности, содержащей римские обозначения цифр от I до IX, получить их десятичные представления.  

### Алгоритм решения

- создается функция inputNumbers, которая формирует последовательность римских чисел
- последовательность создается с помощью Seq.unfold
- программа запрашивает ввод римского числа от I до IX
- если введена пустая строка — последовательность завершается
- если введено корректное римское число — оно добавляется в последовательность
- если введены некорректные данные — выводится сообщение об ошибке и ввод повторяется
- полученная последовательность римских чисел передается в Seq.map
- Seq.map выполняет преобразование римских чисел в десятичные с помощью функции romanToNumber
- полученные десятичные числа выводятся на экран с помощью Seq.iter

## Задача 2

### Текст задачи

Список содержит десятичные цифры. Составить число из чётных цифр.

Ввод цифр осуществляется с клавиатуры.  
Признаком окончания ввода является пустая строка.

### Алгоритм решения

- создается функция inputNumbers, формирующая последовательность введённых пользователем цифр
- последовательность создается с помощью Seq.unfold
- программа запрашивает ввод цифры от 0 до 9
- если введена пустая строка, последовательность завершается
- если введено корректное значение (число от 0 до 9), оно добавляется в последовательность
- если введены некорректные данные, выводится сообщение об ошибке и ввод повторяется
- полученная последовательность цифр передается в функцию Seq.fold
- Seq.fold последовательно перебирает элементы последовательности
- если текущая цифра является чётной, она добавляется к формируемому числу
- добавление происходит по формуле: текущее число умножается на 10 и прибавляется текущая цифра
- если цифра нечётная, она пропускается
- после обработки всей последовательности выводится полученное число
