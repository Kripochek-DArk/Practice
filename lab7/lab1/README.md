# Лабораторная №7

## Задача 1

### Текст задачи

Получить новый файл, уменьшив каждый элемент исходного на 1.

## Задача 2

### Текст задачи

Найти разность первого и максимального элементов.

## Задача 3

### Текст задачи

В файле хранится произвольный текст. Переписать в другой файл строки, начинающиеся с буквы б.

## Задача 4

### Текст задачи

Найти разность первого и максимального элементов файла.

## Задача 5

### Текст задачи

Файл содержит сведения об игрушках: название игрушки, её стоимость в рублях и возрастные границы. Определить стоимость кукол для детей шести лет.

## Задача 6

### Текст задачи

Оставить в списке только первые вхождения одинаковых элементов.

## Задача 7

### Текст задачи

Если у элемента со значением E «соседи» не равны, поменять их местами.

## Задача 8

### Текст задачи

Есть перечень иностранных языков. Работники фирмы могут знать некоторые из них. Для каждого работника известно, какие языки он знает. Определить для каждого языка:

- какие из них знает каждый из работников;
- какие — хотя бы один из работников;
- какие — никто из работников.

## Задача 9

### Текст задачи

Файл содержит текст на русском языке. Определить, с каких букв начинаются слова.

## Задача 10

### Текст задачи

Имеется список учеников разных школ, сдававших экзамен по информатике, с указанием их фамилии, имени, школы и набранного балла. Напишите программу, которая будет определять двух учеников школы № 50, которые лучше всех сдали информатику, и выводить на экран их фамилии и имена.

Если наибольший балл набрали более двух человек, нужно вывести только их количество. Если наибольший балл набрал один человек, а следующий балл набрало несколько человек, нужно вывести только фамилию и имя лучшего. Известно, что информатику сдавали не менее 5 учеников школы № 50.


## Алгоритм решения

- Создан класс File, содержащий статические методы для работы с файлами
- Реализована генерация текстовых файлов с целыми числами (по одному и по несколько в строке)
- Разработаны методы обработки текстовых файлов:

  - изменение значений элементов
  - поиск максимальных и минимальных значений
  - фильтрация строк по заданному условию
- Реализована работа с бинарными файлами:

  - запись чисел
  - чтение и обработка данных
- Созданы классы Toy и ToyCollection для представления структурированных данных
- Реализована XML-сериализация и десериализация объектов
- Разработан метод поиска объектов по заданным параметрам (тип и возраст)

- Создан класс Collections, содержащий методы для работы с коллекциями
- Реализована обработка List:

  - удаление повторяющихся элементов с сохранением первых вхождений
- Реализована работа с LinkedList:

  - изменение значений соседних элементов
- Реализована работа с HashSet:

  - поиск элементов, присутствующих у всех
  - поиск элементов, присутствующих хотя бы у одного
  - поиск элементов, отсутствующих у всех
- Реализована обработка текстового файла:

  - определение первых букв слов
- Реализована обработка данных из файла:

  - выборка по условию
  - поиск максимальных и вторых максимальных значений

- Создан класс Program, объединяющий все решения
- Реализован ввод данных пользователем:

  - списков
  - параметров обработки
  - значений для вычислений
- Организован последовательный вызов всех методов
- Реализован вывод результатов работы программы в консоль
