# Лабораторная №2

## Задача 1

### Текст задачи

На основе списка, содержащего римские обозначения цифр от I до IX, получить их десятичные представления.

### Алгоритм решения

- в функцию передается пустой список
- считывание римских чисел через терминал в список, с проверкой на корректность ввода
- вывод списка римских чисел
- перебирание значений списка через List.map и преобразование римских чисел в десятичные
- вывод списка десятичных числа
### Тестирование

Скриншоты результата работы программы 

## Задача 2

### Текст задачи

Список содержит десятичные цифры. Составить число из чётных цифр.
### Алгоритм решения

- в функцию передается пустой список
- считывание чисел с проверкой на корректность ввода и десятичное ли число
- перебирание значений списка через List.fold и добавление четных чисел в новое число
- вывод нового числа
### Тестирование

Скриншоты результата работы программы
